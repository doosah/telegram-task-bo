КРАТКО:
1. Войдите на https://railway.app через GitHub
2. Нажмите "New Project" → "Deploy from GitHub repo"
3. Выберите ваш репозиторий
4. Добавьте 3 переменные (см. RAILWAY_ПРОСТО.md)
5. В Settings → Start Command введите: python bot.py
6. Готово!
